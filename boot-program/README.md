# 引导程序模块

## 概述

引导程序模块负责系统的启动过程，包括引导加载器、内核加载和系统初始化。

## 功能特性

- **引导加载器**：负责加载内核镜像到内存
- **内核加载器**：解析内核格式并加载到正确位置
- **系统初始化**：初始化硬件和系统服务
- **启动参数解析**：处理内核命令行参数
- **多重启动支持**：支持多种启动配置

## 文件结构

- `boot_loader.h/cpp` - 引导加载器实现
- `kernel_loader.h/cpp` - 内核加载器实现
- `boot_params.h` - 启动参数定义
- `boot_config.h` - 启动配置管理
- `CMakeLists.txt` - 构建配置
